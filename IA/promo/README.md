# BRO – Série de vidéos promotionnelles

Série de scripts pour promouvoir les services **#BRO** (Bot Réponse Ollama / UPlanet IA) en vidéo : avantages et économies d’une **IA locale** inscrite au **commun numérique UPlanet**.

## Objectifs

- Montrer les **avantages** d’une IA locale (vie privée, souveraineté, contexte).
- Chiffrer et illustrer les **économies** (pas d’abonnement, pas de facturation à l’usage, infrastructure partagée).
- Présenter le **commun numérique UPlanet** (CopyLaRadio, ZenCard, mémoire partagée, ORE).

## Fichiers

| Fichier | Durée cible | Thème |
|--------|-------------|--------|
| `01-teaser-pourquoi-bro.md` | 30–45 s | Pourquoi BRO : IA locale + commun. |
| `02-economies.md` | 1–2 min | Économies : coûts évités, infra partagée. |
| `03-vie-privee-souverainete.md` | 1 min | Vie privée et souveraineté des données. |
| `04-fonctionnalites-en-action.md` | 2–3 min | Démo : mémoire, fil, recherche, image, PlantNet. |
| `05-appel-action.md` | 45 s | Devenir sociétaire, rejoindre UPlanet. |

## Formats suggérés

- **Réseaux sociaux** : extraits 30–60 s à partir des scripts 01, 02, 05.
- **YouTube / Peertube** : 01 + 04 (démo) ou playlist complète 01 → 05.
- **Slides / voix off** : chaque script peut servir de trame pour une narration.

## Conventions des scripts

- **[VISUEL]** : idée d’image ou de plan.
- **[VO]** : voix off / narration.
- **[TEXTE]** : titre ou sous-titre à l’écran.
- **Durée** : estimation en secondes entre crochets quand utile.
