🗡️ Jour 1 : Votre Navigateur comme Arme

Astroport transforme votre navigateur.
Aujourd'hui : archivez n'importe quelle vidéo ou musique du web directement sur votre uDRIVE décentralisé.

📥 Super-Pouvoir : Télécharger YouTube

Votre Astroport a installé yt-dlp, l'outil le plus puissant pour archiver le web audiovisuel.

Installez d'abord l'extension Firefox :

🦊 Open With
(addons.mozilla.org → rechercher "Open With")

Cette extension permet d'envoyer l'URL de n'importe quelle page directement à un script local.

⚙️ Configuration Open-With (1/3)

Dans les Options de l'extension Open-With, ajoutez cette entrée :

Nom : Astroport - IPFS

Commande :
~/.zen/Astroport.ONE/ajouter_media.sh

Ce script archive automatiquement le média dans votre uDRIVE IPFS et le publie dans votre TiddlyWiki.

⚙️ Configuration Open-With (2/3) — Vidéos MP4

Nom : Vidéos / MP4 (YouTube, etc.)

Commande : ~/.local/bin/yt-dlp

Arguments :
--cookies-from-browser firefox --download-archive ~/.zen/.yt-dlp.list -f "[height<=480]/best" --write-info-json --no-mtime --playlist-items 1 -o ~/Vidéos/MP4/%(title)s.%(ext)s

Télécharge la vidéo en 480p max, évite les doublons, sauvegarde les métadonnées.

⚙️ Configuration Open-With (3/3) — Musique MP3

Nom : Musique / MP3

Commande : ~/.local/bin/yt-dlp

Arguments :
--cookies-from-browser firefox --download-archive ~/.zen/.yt-dlp.list -x --no-mtime --audio-format mp3 --embed-thumbnail --add-metadata --playlist-items 1 -o ~/Musique/MP3/%(autonumber)s_%(title)s.%(ext)s

Extrait l'audio, intègre la couverture d'album et les métadonnées ID3. Anti-doublon actif.

📂 Où vont vos archives ?

Les fichiers téléchargés sont stockés localement et publiés sur votre uDRIVE IPFS :

  • ~/Vidéos/MP4/ — vos vidéos
  • ~/Musique/MP3/ — votre bibliothèque musicale
  • ~/.zen/.yt-dlp.list — registre anti-doublon

Le script ajouter_media.sh publie ensuite le CID IPFS dans votre TiddlyWiki personnel, accessible depuis n'importe quel appareil de la constellation.

✅ Vos médias vous appartiennent. Personne ne peut les supprimer.

🧪 Test de Votre Installation

Pour tester que tout fonctionne, ouvrez un terminal et tapez :

~/.local/bin/yt-dlp --version

Puis essayez sur une vidéo courte :

~/.local/bin/yt-dlp -f "best[height<=240]" "URL_YOUTUBE"
