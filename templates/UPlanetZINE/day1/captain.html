Cycle economique hebdomadaire

Chaque semaine, le systeme automatiquement :

  • Collecte : MULTIPASS 1 Zen + ZEN Card 4 Zen (+ TVA 20%)
  • Paye le PAF : 1x PAF Armateur + 2x PAF Capitaine
  • Alloue le surplus : 33% Tresorerie + 33% R&D + 33% Actifs

Script : ZEN.COOPERATIVE.3x1-3.sh (declenche a l'anniversaire capitaine)

Accueillir des utilisateurs

Pour creer un MULTIPASS utilisateur :

  • Web : l'utilisateur va sur votre UPassport et scanne le QR
  • CLI : ./tools/make_NOSTRCARD.sh email lang lat lon
  • ZEN Card : creee automatiquement par VISA.new.sh

L'utilisateur recoit un email avec sa ZEN Card (day0) et les ZINE suivants automatiquement.

Restauration et migration

Si un utilisateur arrive d'une autre station :

  • Avec .pass : ./tools/nostr_RESTORE_TW.sh CID_ZIP
  • Sans .pass (meme UPlanet) : ./tools/nostr_RESTORE_TW.sh CID_UPLANET
  • Le cashback G1 est renvoye automatiquement (moins 1 G1 primo)
  • L'historique secret.june (parts cooperatives) est preserve

Si UPlanet differente : .pass obligatoire, pas de cashback.
