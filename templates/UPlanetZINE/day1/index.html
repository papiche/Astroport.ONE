Jour 1 : Votre espace numerique est actif

Votre ZEN Card fonctionne. Voici ce qui tourne deja pour vous, 24h/24.

Ce qui est actif maintenant

  • Votre identite souveraine : une cle unique qui vous identifie sur Nostr, la blockchain G1 et IPFS. Aucun mot de passe a retenir.
  • Votre espace de stockage (uDRIVE) : 128 Go prives, accessibles depuis n'importe quel appareil.
  • Votre journal automatique : Bot Astro collecte les messages de vos amis et cree un journal quotidien pour vous.
  • Votre wallet ZEN : vous commencez avec la primo TX. Chaque like que vous recevez sur vos publications = 1 Zen.

Comment ca marche au quotidien

1. Vous publiez sur Nostr (via Coracle, Amethyst, ou tout client Nostr).

2. Vos amis voient 100% de vos messages, dans l'ordre chronologique. Pas d'algorithme.

3. Les likes que vous recevez se transforment automatiquement en Zen sur votre wallet.

4. Bot Astro vous cree un journal avec les publications de votre constellation (amis + amis d'amis).

Tout est automatique. Vous n'avez rien a configurer.

Votre redevance : 5 Zen / semaine

Chaque semaine, votre wallet est preleve automatiquement :

  • 1 Zen MULTIPASS : acces aux services de base
  • 4 Zen ZEN Card : cooperateur, NextCloud, droit de vote

Votre compte a demarre a 0 Zen. Si dans 7 jours vous n'avez pas au moins 5 Zen, votre compte sera desactive (sauf Capitaine).

Solution rapide : publiez 5 messages qui recueillent chacun 1 like. Ou devenez Parrain (50 euros/an = plus de prelevements).

Ou vont les Zen ?

Chaque Zen collecte est reparti automatiquement par le code (pas par un humain) :

  • 1/3 Tresorerie : reserve de fonctionnement pour la perennite
  • 1/3 R&D : developpement des logiciels libres
  • 1/3 Biens Communs : acquisition de terres, forets, infrastructure

L'Armateur (hebergeur) et le Capitaine (operateur) sont defrayes par cette meme economie.

Transparence totale : toutes les transactions sont visibles sur la blockchain G1.

Vos donnees vous appartiennent

  • Zero tracking : aucune publicite, aucun profilage.
  • Zero censure : vos cles cryptographiques garantissent que personne ne peut supprimer vos publications.
  • Portable : si vous changez de station, vos donnees migrent avec vous (MULTIPASS + uDRIVE).
  • Recuperable : votre identite est protegee par le secret de Shamir (SSSS 2/3). Meme si vous perdez votre carte, vous pouvez recuperer votre compte.

Prochaines etapes

  • Installez un client Nostr : Coracle (web) ou Amethyst (Android) pour publier.
  • Suivez des amis : chaque ami que vous suivez etend votre constellation N2.
  • Publiez votre premier message : dites bonjour, gagnez vos premiers Zen.
  • Invitez un ami : le reseau grandit naturellement par les amis d'amis.
