📸 Jour 1 — Synchro automatique des photos

Depuis aujourd'hui, vos photos sont sauvegardées automatiquement sur votre capsule privée.
Plus de "Stockage Google Photos plein".

📱 Activer Import automatique — Android

  1. Ouvrez l'app Nextcloud
  2. Menu hamburger (≡) → Import automatique
  3. Activez la bascule "Import automatique"
  4. Sélectionnez les dossiers à synchroniser (DCIM/Camera)
  5. Choisissez "Télécharger uniquement en WiFi" (recommandé)

📖 Détail complet : pad.p2p.legal/Smartphone2NextCloud

🍎 Activer Import automatique — iPhone

  1. Ouvrez l'app Nextcloud
  2. Appuyez sur votre avatar (coin haut-droit)
  3. → Paramètres → Import automatique des photos
  4. Activez et accordez les permissions demandées
  5. Sélectionnez "WiFi seulement" pour économiser votre data

La première synchro peut durer longtemps si vous avez beaucoup de photos. Laissez charger en WiFi.

🔋 Conseils pour la batterie

  • ✅ WiFi seulement : évite la surconsommation data
  • ✅ En charge seulement : option disponible dans les paramètres
  • ✅ La nuit : configurez la synchro en nocturne pour ne pas perturber votre journée
  • ⚠️ Première synchro : comptez 1-2h pour 1000 photos en WiFi

Une fois la première synchro faite, les nouvelles photos sont uploadées en secondes.

📺 Et les vidéos ?

Les vidéos sont synchronisées avec les photos si vous avez sélectionné le dossier DCIM complet.

Pour les vidéos téléchargées (YouTube, etc.), utilisez yt-dlp :

~/.local/bin/yt-dlp -f "best[height<=480]" URL_VIDEO

Les fichiers téléchargés peuvent ensuite être copiés dans votre capsule NextCloud via le client desktop.
