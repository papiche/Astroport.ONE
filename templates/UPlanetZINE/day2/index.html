Jour 2 : Le Jardin Numerique

Vous connaissez les AMAP Paysannes ? Un groupe de citoyens finance un paysan, partage la recolte, possede le jardin ensemble.

Le G1FabLab, c'est pareil. Mais pour Internet. Une AMAP Numerique ou chacun co-possede l'infrastructure au lieu de louer chez Google.

AMAP Paysanne / AMAP Numerique

AMAP Paysanne :

  • Vous financez le terrain et les semences
  • Le paysan cultive et vous livre les fruits
  • Les semences sont libres et reproductibles

AMAP Numerique (G1FabLab) :

  • Vous financez les serveurs et le code open source
  • Le Capitaine maintient les services et vous les rend accessibles
  • Le code est libre et reproductible : chacun peut le replanter

MULTIPASS : achetez les fruits

En tant que locataire MULTIPASS, vous achetez les fruits du jardin :

  • 1 Zen/semaine : acces aux services (identite, uDRIVE, IA, journal N2)
  • Vous consommez sans posseder le terrain
  • Si vous arretez de payer, vous n'avez plus de fruits

C'est comme acheter ses legumes au marche : pratique, mais vous ne possedez pas le champ.

ZEN Card : possedez le terrain

En tant que cooperateur ZEN Card, vous co-possedez le jardin :

  • +4 Zen/semaine : vous financez le terrain (serveurs), les outils (logiciels) et les semences (R&D)
  • 128 Go NextCloud : votre propre parcelle dans le jardin
  • Droit de vote : vous decidez ce qu'on plante (1 personne = 1 voix)
  • Foret comestible : 1/3 des contributions achete de vraies terres et forets pour le collectif

C'est comme une AMAP : vous possedez le champ, les outils et les graines. Les graines sont libres : chacun peut les partager.

Parrain : plantez un arbre

Le Parrainage remplace la redevance hebdomadaire par une contribution annuelle :

  • Satellite (50 euros/an) : vous plantez un arbre (part d'un Raspberry Pi). NextCloud 128 Go + vote + plus de redevance hebdomadaire.
  • Constellation (540 euros/3 ans) : vous plantez un verger entier (part d'un serveur GPU). Acces IA video + priorite services.

C'est un don au collectif, pas un achat. Vous financez le jardin pour que tous les jardiniers en profitent.

Armateur et Capitaine

Armateur = le proprietaire du terrain qui prete sa parcelle :

  • Heberge un serveur chez lui (fibre + electricite)
  • Recoit 14 Zen/semaine de redevance d'hebergement (le "loyer du champ")

Capitaine = le paysan qui cultive le jardin :

  • Installe, maintient et opere la station
  • Recoit 28 Zen/semaine de retribution (le "salaire du jardinier")

Armateur et Capitaine peuvent facturer en euros au G1FabLab via Open Collective.

La regle des 3 tiers

Chaque contribution est repartie automatiquement par le code :

  • 1/3 Tresorerie : la terre et l'eau (electricite, bande passante, materiel de remplacement)
  • 1/3 R&D : les semences libres reproductibles (code open source, nouvelles fonctionnalites)
  • 1/3 Biens Communs : la foret comestible (acquisition de vraies terres et forets pour tous les cooperateurs)

Transparence totale : toutes les transactions sont sur la blockchain G1, verifiables en temps reel.

Semences libres, jardin partage

Tout le code est open source (AGPL-3.0).

Comme des graines libres reproductibles : chacun peut les replanter, les ameliorer, les partager. Plus il y a de jardiniers, plus le jardin est productif.

Demain : le Protocole N2, votre reseau social sans algorithme.
