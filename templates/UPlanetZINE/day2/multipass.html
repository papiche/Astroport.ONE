🧠 Qu'est-ce que #BRO ?

#BRO (Brain Robot Online) est l'IA collective de la constellation Astroport :

  • Locale : tourne sur votre machine via Ollama. Pas de données envoyées à OpenAI.
  • Partagée : les Capitaines programment ensemble ce qu'il peut faire pour vous.
  • Souveraine : financée collectivement, accessible à tous les membres.

La puissance GPU des stations de la constellation est mutualisée. Vous bénéficiez du collectif même sans GPU dédié.

🎨 Ce que #BRO peut faire pour vous

  • 🖼️ Génération d'images : créez des visuels pour vos publications Nostr
  • 📝 Rédaction d'articles : publiez du contenu documenté sur votre blog décentralisé
  • 🔍 Recherche sémantique : interrogez le savoir de votre constellation
  • 🌙 Travail nocturne : #BRO travaille pendant que vous dormez, collecte et analyse

Le collectif de Capitaines programme en permanence de nouvelles capacités pour #BRO.

🐝 L'Intelligence de l'Essaim

Contrairement aux IA centralisées, #BRO est un essaim d'intelligences :

  • Chaque station Astroport exécute son propre modèle local
  • Les résultats sont agrégés via IPFS (pas de point central)
  • Chaque Capitaine peut spécialiser son nœud (images, texte, code...)

C'est la différence fondamentale avec ChatGPT : ici, c'est votre intelligence collective, pas celle d'une corporation.

🛠️ Stack #BRO sur votre machine

Votre Astroport a installé :

  • Ollama : moteur LLM local (ollama list)
  • Stable Diffusion : génération d'images (via Docker)
  • Qdrant : base de vecteurs pour la recherche sémantique

Pour installer le stack IA complet (GPU recommandé) :

~/.zen/Astroport.ONE/install/install-ai-company.docker.sh
