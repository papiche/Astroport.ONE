📇 Jour 2 — Contacts & Agenda : quittez Google

Vos contacts et votre agenda sont actuellement chez Google ou Apple.
Aujourd'hui, on les rapatrie sur votre capsule NextCloud.

📤 Exporter vos contacts Google

  1. Ouvrez contacts.google.com sur votre ordinateur
  2. Menu ≡ → Exporter
  3. Sélectionnez tous vos contacts
  4. Format : vCard (.vcf)
  5. Cliquez Exporter → fichier téléchargé

Guide complet : pad.p2p.legal/Smartphone2NextCloud

📥 Importer dans NextCloud

  1. Connectez-vous à https://cloud._EARTHCID_
  2. Cliquez sur l'icône Contacts (👤 en haut)
  3. Cliquez sur les 3 points (⋮) → Importer
  4. Sélectionnez votre fichier .vcf
  5. Vos contacts apparaissent dans NextCloud ✅

Si vos contacts sont sur iPhone : Réglages → [Votre nom] → iCloud → Contacts → Exporter vCard.

📱 Synchroniser les contacts sur smartphone

Android — avec DAVx⁵ (F-Droid) :

  1. Installez DAVx⁵ depuis F-Droid
  2. URL : https://cloud._EARTHCID_/remote.php/dav
  3. Vos identifiants NextCloud
  4. Activez la synchro Contacts + Agenda

DAVx⁵ est l'outil recommandé : f-droid.org

📅 Migrer votre agenda Google Calendar

  1. Allez sur calendar.google.com → Paramètres (⚙️)
  2. Sélectionnez votre agenda → Exporter
  3. Téléchargez le fichier .ics
  4. Dans NextCloud → Agenda → Import
  5. Sélectionnez votre fichier .ics ✅

Avec DAVx⁵, votre agenda se synchronise automatiquement entre votre téléphone et NextCloud.

🔮 La différence concrète

  • ❌ Google Contacts : analyse vos contacts pour ciblage pub, partagé avec les apps
  • ❌ Google Calendar : vos RDV analysés pour vous proposer des services
  • ✅ NextCloud : vos contacts + agenda sur votre machine, chiffrés, accessibles hors-ligne

La synchronisation est automatique et bidirectionnelle : ajoutez un contact sur votre téléphone → il apparaît dans NextCloud, et vice-versa.

✅ Vérification finale

Après migration, vérifiez que tout est bien synchro :

  • Ouvrez NextCloud web → Contacts : vos contacts sont là ✓
  • Sur votre téléphone : un nouveau contact Google disparu ? Vérifiez DAVx⁵
  • Agenda : Créez un événement sur le web → visible sur le téléphone en quelques secondes
