Jour 4 : Passez a l'action

Vous avez teste le reseau pendant 3 jours. Maintenant, voici comment passer de simple utilisateur a co-proprietaire de l'infrastructure.

3 niveaux d'engagement. Choisissez celui qui vous convient.

Niveau 1 : Parrainage (50 euros/an)

Vous financez l'achat de materiel pour le reseau :

  • 128 Go NextCloud : synchronisation smartphone, fichiers, contacts, calendrier
  • Droit de vote : 1 personne = 1 voix sur les decisions de la cooperative
  • Plus de redevance hebdomadaire : votre contribution annuelle couvre tout
  • Support technique : accompagnement personnalise

Contribution via opencollective.com/monnaie-libre

Niveau 2 : Armateur (hebergement)

Vous avez la fibre et un espace chez vous ? Hebergez un noeud Astroport :

  • Materiel fourni par les Parrains (mini-PC ou RPi5)
  • 14 Zen/semaine de redevance d'hebergement
  • Contrat simple : convention d'hebergement avec le G1FabLab
  • Vous fournissez : espace, electricite, fibre, redemarrage si besoin

Le materiel reste proprietaire du collectif. Declarez vos revenus en BIC ou BNC.

Niveau 3 : Capitaine (operation)

Vous etes technicien ? Operez et maintenez des stations :

  • 28 Zen/semaine de retribution + excedent de votre essaim
  • Missions : installation, maintenance, accueil utilisateurs, validation SSSS
  • Independance : auto-entrepreneur, pas salarie (SIRET requis)
  • Facturation : via Open Collective, virement bancaire mensuel

Formation et documentation completes fournies.

Le service de remboursement

Les Zen se convertissent en euros via le G1FabLab :

  • 1. Le protocole mesure votre activite (hebergement, maintenance)
  • 2. Vous emettez une facture en euros au G1FabLab
  • 3. Le collectif valide et declenche le virement bancaire
  • 4. Les Zen correspondants sont soldes

Vous ne declarez que les euros recus, pas les Zen. Le fait generateur fiscal est le virement.

Pourquoi c'est different

  • Pas d'actionnaire : le G1FabLab est une cooperative (SCIC), pas une startup.
  • Pas de dividende : les excedents sont reinvestis dans l'infrastructure commune.
  • Transparence radicale : comptabilite blockchain publique, verifiable en temps reel.
  • Code open source : les regles du jeu sont dans le code (AGPL-3.0), auditables par tous.

Analogie : Google = acheter vos legumes a Carrefour. UPlanet = cotiser a un jardin partage dont vous partagez la recolte.

Partagez avec vos amis

Le reseau UPlanet grandit par les amis d'amis. Chaque personne que vous invitez :

  • Etend votre constellation N2 : plus d'amis = plus de contenu = plus de Zen
  • Renforce le reseau : plus de noeuds = plus de resilience
  • Finance l'infrastructure : chaque redevance d'usage renforce le commun
  • Construit un patrimoine : le tiers Biens Communs achete des terres et forets pour tous

L'Internet de confiance se construit brique par brique, entre amis.
