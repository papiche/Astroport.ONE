🧠 Qu'est-ce que #BRO ?

#BRO (Brain Robot Online) est l'IA collective de la constellation Astroport :

  • 🏠 Local : tourne sur chaque station via Ollama — zéro cloud externe
  • 🌐 Essaim : les stations partagent leur puissance de calcul la nuit
  • 🔒 Privé : vos questions ne quittent pas la constellation
  • 📈 Évolutif : plus il y a de Capitaines, plus l'essaim est puissant

API locale Ollama : http://127.0.0.1:11434

💬 Commander #BRO depuis Nostr

La façon la plus simple d'interagir : publiez sur Nostr avec le tag #BRO

#BRO résume mes derniers posts Nostr

#BRO crée une image de : un jardin
coopératif avec des panneaux solaires

#BRO article : les bienfaits de la
Monnaie Libre Ğ1

#BRO rapport hebdomadaire de ma station

Accédez à votre client Nostr : _USPOT_/nostr

⚙️ Accès direct à l'API Ollama

Pour les plus curieux, l'API est accessible directement :

# Lister les modèles disponibles
curl http://127.0.0.1:11434/api/tags

# Poser une question
curl http://127.0.0.1:11434/api/generate \
  -d '{
    "model": "gemma3",
    "prompt": "Bonjour BRO, comment fonctionne IPFS ?",
    "stream": false
  }'

Si Ollama n'est pas installé sur cette station, le modèle sera celui de la constellation la plus proche.

🎨 Génération d'images

#BRO peut générer des images pour illustrer vos posts Nostr :

Sur Nostr, publiez :

#BRO crée une image de :
[votre description en français]

Exemples :
- "un village numérique coopératif"
- "une forêt avec des antennes IPFS"
- "portrait d'un Capitaine Astroport"

💡 Les images générées sont stockées dans votre uDRIVE IPFS et publiables directement sur Nostr.

  • 🤝 Échange de connaissances entre les nœuds N1/N2

Plus la constellation grandit, plus l'essaim devient intelligent. Chaque Capitaine certifié renforce le réseau.
