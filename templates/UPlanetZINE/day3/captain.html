Captain Jour 3 : Votre infrastructure

Votre station tourne sur 3 conteneurs Docker. Voici comment verifier, maintenir et activer chaque service.

Les 3 conteneurs Docker

  • astroport : IPFS + strfry + UPassport + APIs (toujours actif)
  • npm : Nginx Proxy Manager — SSL et reverse proxy (toujours actif)
  • nextcloud : NextCloud AIO 128 Go (optionnel, profil full)

Commandes de base :

docker compose ps
docker compose logs -f astroport
docker compose logs -f npm

Nginx Proxy Manager (NPM)

NPM gere le SSL et les sous-domaines :

  • astroport.DOMAIN : Station Map (port 12345)
  • ipfs.DOMAIN : IPFS Gateway (port 8080)
  • relay.DOMAIN : NOSTR Relay (port 7777)
  • u.DOMAIN : UPassport API (port 54321)
  • cloud.DOMAIN : NextCloud (port 8001)

Admin NPM : http://127.0.0.1:81 (local uniquement)

Verifier que tout fonctionne

# Sante des services
curl -sf http://127.0.0.1:54321/health
curl -sf http://127.0.0.1:81
curl -sf http://127.0.0.1:8080/ipfs/QmTest

# Logs en temps reel
docker compose logs -f --tail=50

# Redemarrer un service
docker compose restart astroport
docker compose restart npm

En bare metal : sudo systemctl status astroport upassport strfry ipfs

Activer NextCloud (128 Go)

NextCloud offre le stockage synchronise aux cooperateurs ZEN Card :

# Activer NextCloud
docker compose --profile full up -d

# Admin NextCloud AIO
https://127.0.0.1:8443

# Verifier
docker compose logs -f nextcloud
  • Premiere connexion : configurez le domaine cloud.VOTREDOMAINE
  • NPM cree automatiquement le proxy cloud.DOMAIN → port 8001
  • Chaque ZEN Card = 1 compte NextCloud 128 Go

SSL et certificats

  • Domaine public : Let's Encrypt automatique via NPM
  • Domaine local (.local, 192.168.x) : auto-signe openssl (10 ans)
  • copylaradio.com : gere centralement (ne pas modifier)

Le setup NPM est automatique au premier demarrage (setup_npm.sh). Les certificats se renouvellent seuls.

Token NPM : ~/.zen/nginx-proxy-manager/data/.api_token

Smartphone → NextCloud

Pour les cooperateurs ZEN Card, configurez la synchro smartphone :

  • Android : installer NextCloud (F-Droid ou Play Store)
  • Serveur : https://cloud.VOTREDOMAINE
  • Synchro auto : photos, contacts, calendrier

Guides : pad.p2p.legal/NextCloud et pad.p2p.legal/Smartphone2NextCloud
