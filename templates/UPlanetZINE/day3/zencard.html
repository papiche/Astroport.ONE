📒 Jour 3 — Contacts & Agenda libérés

Vos contacts et votre agenda sont chez Google ou Apple.
Aujourd'hui, vous les rapatriez dans votre capsule. Une seule installation : DAVx⁵.

📲 Installer DAVx⁵ sur Android

DAVx⁵ est l'app qui synchronise contacts (CardDAV) et agenda (CalDAV) avec NextCloud :

  1. Ouvrez F-Droid (recommandé) ou le Play Store
  2. Recherchez : DAVx⁵
  3. Installez et ouvrez l'app
  4. Appuyez sur "+" → "Connexion avec une URL et identifiant"

📖 Guide détaillé : pad.p2p.legal/Smartphone2NextCloud

🔗 Connecter DAVx⁵ à votre capsule

Dans DAVx⁵, entrez :

https://cloud._EARTHCID_

  • Identifiant : celui fourni par votre Capitaine
  • Mot de passe : votre mot de passe NextCloud

DAVx⁵ détecte automatiquement CardDAV (contacts) et CalDAV (agenda). Cochez les deux, puis "Créer le compte".

⚠️ Si vous avez changé votre mot de passe NextCloud hier, utilisez le nouveau.

📤 Exporter vos contacts Google

Avant de couper Google, exportez :

  1. Sur ordinateur : contacts.google.com
  2. Menu ≡ → Exporter → format vCard (.vcf)
  3. Importez ce fichier .vcf dans votre NextCloud :
    cloud._EARTHCID_ → Contacts → ⋮ → Importer
  4. DAVx⁵ les synchronise automatiquement sur votre téléphone

Même procédure pour iCloud : Réglages → [votre nom] → iCloud → Contacts → Exporter vCard.

📅 Migrer votre agenda

Pour l'agenda Google :

  1. Sur ordinateur : calendar.google.com
  2. Paramètres ⚙️ → votre agenda → Exporter → .ics
  3. Dans NextCloud : app Agenda → Importer le fichier .ics
  4. DAVx⁵ synchronise votre agenda NextCloud sur le téléphone

💡 Vous pouvez garder Google en lecture seule le temps de la transition. Coupez-le quand vous êtes prêt.

🔒 Ce que vous gagnez

  • ❌ Google Contacts analyse vos relations pour cibler des publicités
  • ❌ iCloud : vos données sous juridiction américaine (CLOUD Act)
  • ✅ Votre capsule : contacts et agenda chez votre Capitaine, chiffrés, sous votre contrôle
  • ✅ Sync universelle : Android, iPhone, Linux, Mac, Windows — via DAVx⁵ + Thunderbird

Sur Linux/Mac/Win : Thunderbird + extension TbSync pour sync CardDAV/CalDAV.
