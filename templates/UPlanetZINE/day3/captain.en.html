Captain Day 3: Your infrastructure

Your station runs on 3 Docker containers. Here is how to check, maintain and activate each service.

The 3 Docker containers

  • astroport: IPFS + strfry + UPassport + APIs (always active)
  • npm: Nginx Proxy Manager — SSL and reverse proxy (always active)
  • nextcloud: NextCloud AIO 128 GB (optional, full profile)

Basic commands:

docker compose ps
docker compose logs -f astroport
docker compose logs -f npm

Nginx Proxy Manager (NPM)

NPM manages SSL and subdomains:

  • astroport.DOMAIN: Station Map (port 12345)
  • ipfs.DOMAIN: IPFS Gateway (port 8080)
  • relay.DOMAIN: NOSTR Relay (port 7777)
  • u.DOMAIN: UPassport API (port 54321)
  • cloud.DOMAIN: NextCloud (port 8001)

NPM Admin: http://127.0.0.1:81 (local only)

Verify everything is working

# Service health
curl -sf http://127.0.0.1:54321/health
curl -sf http://127.0.0.1:81
curl -sf http://127.0.0.1:8080/ipfs/QmTest

# Real-time logs
docker compose logs -f --tail=50

# Restart a service
docker compose restart astroport
docker compose restart npm

On bare metal: sudo systemctl status astroport upassport strfry ipfs

Activate NextCloud (128 GB)

NextCloud provides synchronized storage for ZEN Card cooperators:

# Activate NextCloud
docker compose --profile full up -d

# NextCloud AIO Admin
https://127.0.0.1:8443

# Verify
docker compose logs -f nextcloud
  • First connection: configure the domain cloud.YOURDOMAIN
  • NPM automatically creates the proxy cloud.DOMAIN -> port 8001
  • Each ZEN Card = 1 NextCloud account 128 GB

SSL and certificates

  • Public domain: Let's Encrypt automatic via NPM
  • Local domain (.local, 192.168.x): self-signed openssl (10 years)
  • copylaradio.com: managed centrally (do not modify)

The NPM setup is automatic on first start (setup_npm.sh). Certificates renew themselves.

NPM Token: ~/.zen/nginx-proxy-manager/data/.api_token

Smartphone -> NextCloud

For ZEN Card cooperators, configure smartphone sync:

  • Android: install NextCloud (F-Droid or Play Store)
  • Server: https://cloud.YOURDOMAIN
  • Auto sync: photos, contacts, calendar

Guides: pad.p2p.legal/NextCloud and pad.p2p.legal/Smartphone2NextCloud
