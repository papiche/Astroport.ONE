Votre contribution de Parrain (Satellite ou Constellation) touche a sa fin. Sans renouvellement, vous passerez en mode locataire avec redevance hebdomadaire (4 Zen/semaine pour la ZEN Card).

Pourquoi renouveler ?

Imaginez un jardin partage. Votre parrainage, c'est la terre, les outils et les graines. Sans terre, pas de recolte. Sans outils, pas de culture. Votre contribution finance :

  • Le terrain (serveurs, disques) : l'infrastructure qui fait tourner vos services
  • Les outils (logiciels libres) : les applications que vous utilisez chaque jour
  • Les semences (R&D) : le developpement de nouvelles fonctionnalites

Et comme des semences libres reproductibles, tout le code est open source : chacun peut le replanter, l'ameliorer, le partager. C'est un jardin qui grandit avec chaque jardinier.
