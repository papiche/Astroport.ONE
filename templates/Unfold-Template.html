Scrolling speed(scroll/sec):

The value is scrolling speed. A unit is scroll / sec.
Max is 3000, and min is 1.

Enable transmission scrolling:

If enabled, you can change scrolling speed per gear. The gears are Slow / Middle / Fast. The gears are cycled between Slow, Middle, Fast by single/double-clicking browser icon.

Scrolling stop by click:

If enabled the option, scrolling is stopped when you click the scrolling window.

Stop scrolling by cursor hovering over something:

If enabled the option, scrolling is stopped temporally when mouse cursor is hovering over something.

Keyboard shortcut for single-click action:

Fire a single-click action by keyboard shortcut

  • Start scrolling when window is stopped
  • Stop scrolling when window is scrolling
  • Close option modal window when the window is opened
The keybind should be a combination of keycodes concatenated with the '+' sign.
Ex: Ctrl+Shift+Insert.
Check out the link for the keycodes: Key combinations
Notice: Some key combinations are reserved by the browser and can't be reassigned to this.

Stop scrolling on focus out:

If enabled the option, scrolling is stopped when focus for the browser is moved to another application or browser window.

Disable double click action:

  • start/stop scrolling is immediately

Restore scrolling from switch back:

If this option is enabled, scrolling which stopped by tab changing restarts when user is back to a tab that was scrolling.

Stop when bottom of window:

If enabled, scrolling is stopped when window position is reached at bottom.
